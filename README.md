## Install
Clone this project

```bash
> git clone https://github.com/Stalker-moment/Web-BountyGenerator/
> cd Web-BountyGenerator
```

Install the dependencies:

```bash
> npm install
```

### Usage
1. run the Code

 Hard Mode

```bash
> npm start
```

 Nodemon 

```bash
> npm test
```
 or
```bash
> node pickernumber.js
```

2. force stop the code
```bash
> ctrl + c
```
